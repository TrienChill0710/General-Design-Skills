Chào mừng bạn đến với Slack Summarizer

Trợ lý ảo giúp bạn đọc, lọc, tóm tắt và hỏi đáp các thông tin từ các kênh Slack một cách thông minh.

1

Cấu hình API Key

Mở Cấu hình hệ thống để thiết lập Gemini API Key nhằm kích hoạt tính năng tóm tắt bằng AI.

2

Nạp dữ liệu Slack

Đồng bộ trực tiếp qua Slack API hoặc tải lên file ZIP export từ Workspace của bạn.

3

Chọn kênh & Tóm tắt

Chọn kênh trò chuyện, chọn khoảng thời gian và xem kết quả phân tích tức thì từ AI.
